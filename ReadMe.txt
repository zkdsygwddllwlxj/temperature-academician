适用环境：window10（已验证）
部署步骤：
    1.先替换文件startps.vbs里的路径，用文件main.ps1的绝对路径代替
    2.然后把文件startps.vbs剪切到系统的开机启动目录里：C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp
    3.最后重启机器

使用方法：
    1.浏览器打开文件index.html可以查看实时温度
    2.打开result.txt可以查看每天记录的温度（每日24点更新）
